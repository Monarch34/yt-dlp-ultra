# Contributing to yt-dlp-ultra

Thank you for your interest in contributing to the yt-dlp-ultra project! We welcome contributions from everyone. Please read the guidelines below to understand how to get involved.

## How to Contribute

1. **Fork the Repository**: Start by forking the repository on GitHub to your own account.

2. **Clone Your Fork**: Clone your forked repository to your local machine.

   ```bash
   git clone https://github.com/your_username/yt-dlp-ultra.git
   cd yt-dlp-ultra
   ```

3. **Create a Branch**: Create a new branch for your feature or bug fix.

   ```bash
   git checkout -b your-feature-branch
   ```

4. **Make Your Changes**: Implement your changes and ensure that your code is well-tested.

5. **Commit Your Changes**: Commit your changes with a clear message explaining what you did.

   ```bash
   git commit -m "Add your message here"
   ```

6. **Push Your Changes**: Push your changes to your forked repository.

   ```bash
   git push origin your-feature-branch
   ```

7. **Create a Pull Request**: Go to the [original repository](https://github.com/Monarch34/yt-dlp-ultra) and create a pull request. Provide a clear description of your changes.

## Code of Conduct

Please be respectful to other contributors and maintainers. We expect all contributors to follow the [Code of Conduct](CODE_OF_CONDUCT.md).

## Additional Information

- Please ensure your code follows the existing style guidelines.
- If you're fixing a bug, please link to the issue in your pull request.

Thank you for contributing! We appreciate your help in improving yt-dlp-ultra!
