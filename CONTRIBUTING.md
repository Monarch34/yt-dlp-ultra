# Contributing to yt-dlp-ultra

Thank you for considering contributing to yt-dlp-ultra! This project is a Docker wrapper for the original yt-dlp project, which is an open-source video downloader.

## Credit
This project credits the original yt-dlp developers for their hard work. If you are interested in contributing back to the original yt-dlp project as well, feel free to check it out.

## How to Contribute
- Read the [Code of Conduct](CODE_OF_CONDUCT.md).
- Check the [open issues](issues).
- Fork the repository and make your changes.
- Submit a pull request! Your contributions are welcome!
